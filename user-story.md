## User Story 1 (mới)
**As a** user  
**I want** to register an account  
**So that** I can access personalized features  

---

## User Story 2 (tồn đọng)
**As a** user  
**I want** to log in securely  
**So that** my personal data is protected  

---

## User Story 3 (nợ kỹ thuật)
**As a** developer  
**I want** to refactor authentication code  
**So that** the system is easier to maintain  

---

## User Story 4 (tạm dừng)
**As a** product owner  
**I want** to pause the notification feature  
**So that** we can redesign the user experience  

---

## User Story 5 (mới)
**As a** user  
**I want** to reset my password  
**So that** I can regain access if I forget it  

---

## User Story 6 (tồn đọng)
**As a** admin  
**I want** to manage user roles  
**So that** I can control system permissions  

---

## User Story 7 (nợ kỹ thuật)
**As a** developer  
**I want** to improve database indexing  
**So that** queries run faster  

---

## User Story 8 (tạm dừng)
**As a** user  
**I want** to use dark mode  
**So that** the app is easier to use at night
